# ECS Stack

This directory contains the infrastructure-as-code for the ECS (Elastic Container Service) stack.

## Purpose

The ECS stack defines the container orchestration infrastructure including:
- ECS clusters
- Task definitions
- Services
- Load balancers
- Auto-scaling configurations

## Environments

This stack is deployed to the following environments:
- development
- staging

## Configuration

This stack is configured in the `colonia.yaml` file at the repository root.
