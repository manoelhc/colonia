# VPC Stack

This directory contains the infrastructure-as-code for the VPC (Virtual Private Cloud) stack.

## Purpose

The VPC stack defines the network infrastructure including:
- Virtual Private Cloud
- Subnets
- Route tables
- Internet gateways
- NAT gateways

## Environments

This stack is deployed to the following environments:
- development
- staging

## Configuration

This stack is configured in the `colonia.yaml` file at the repository root.
